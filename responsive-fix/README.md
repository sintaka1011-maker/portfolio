# スマホ表示崩れの修正（レスポンシブ対応）

## 概要
スマートフォン表示時にレイアウトが崩れ、
横スクロールが発生していたページを想定し、
レスポンシブ対応の改修を行いました。

※ 実務を想定した架空案件として制作しています。

## 修正前の問題点
- コンテンツ幅が固定されており、スマホ表示で横スクロールが発生
- ボタンが小さく、タップしづらい

## 修正内容
- viewport の設定
- width 指定を max-width に変更
- padding 調整による余白改善
- メディアクエリを用いたスマホ対応
- ボタンサイズの調整

## 使用技術
- HTML
- CSS（レスポンシブ対応）

## 工夫した点
既存HTML構造を極力変更せず、
CSSの修正のみで対応することを意識しました。

## 修正後の状態
スマートフォン表示でも横スクロールが発生せず、
快適に操作できるレイアウトになりました。

※ before.png / after.png にて表示の違いを確認できます。
