## 概要
ユーザーに行動を促すCTA（Call To Action）ボタンを想定し、視認性と操作性の向上を目的に改修を行いました。

## 修正前の問題点
- ボタンが小さく、押せる要素だと分かりにくい
- ホバー・フォーカス時の視覚的な変化がない
- ボタンを連続してクリックできてしまう
- 押下後の状態が分かりにくい

## 修正内容
- ボタンサイズと余白の調整
- hover / active / focus の状態を追加
- スマートフォン操作を想定したスタイルに変更
- クリック時にボタンを一時的に無効化
- 文言を「送信中...」に変更
- 一定時間経過後に元の状態へ戻す処理を追加

## 使用技術
- HTML
- CSS（UI改善・擬似クラス）
- JavaScript（クリック制御・状態管理）

## 工夫した点
- 既存デザインを大きく変更せず、操作性の改善にフォーカスしました。
- 元のボタン文言を変数として保持し、状態を戻す際に再利用できるようにしました。
